# Security Policy

MiniCAD is a small, early-stage hobby project. It's provided **as-is, with no warranty**
and no formal security process or guaranteed response times.

That said, if you notice something that looks like a security problem, it's still helpful
to flag it:

- Please [open a GitHub issue](https://github.com/Fxbixn03/MiniCAD/issues/new/choose)
  describing what you found and how to reproduce it.
- Since this is a desktop hobby project with no servers or user accounts, there's no
  sensitive data at risk — so a normal public issue is perfectly fine.

Thanks for helping keep the project healthy. 🙏
